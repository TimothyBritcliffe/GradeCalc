# CLI To-Do List Application

A basic C# .NET MAUI application for converting number grades (input) into letter grades defined by the Thompson Rivers University (TRU) grade scale.
Grade Scale: https://www.tru.ca/distance/services/policies/newgrading.html#:~:text=Undergraduate%20Academic/Career/Developmental%20Programs


## Purpose
The objective of this application was to act as a demo for me to get more familiar with .NET MAUI application development.


## Getting Started
1. Clone this repository.
2. Open the solution in **JetBrains Rider** (or an IDE of your choosing, however I developed this in Rider).
3. Run the application.
